
# Astore (Spring Boot)

Система управления интернет-магазином, построенная на базе Spring Framework. Проект предназначен для управления каталогом товаров, включая создание, чтение, обновление и удаление продуктов.

## Технологический стек

- **Runtime:** Java 21
- **Framework:** Spring Boot 4.0.2
- **Build Tool:** Maven

## Функциональные возможности

- ✅ Получение всех товаров
- ✅ Получение товара по ID
- ✅ Фильтрация товаров по категории
